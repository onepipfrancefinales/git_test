Tests aux lecteurs d'écrans Formation développeur

Structure de la page

Avec NVDA :

  • Naviguer entre les régions : D et Shift D
  • Afficher la liste des régions : inser F7

Avec JAWS :

  • Naviguer entre les régions : R et Shift R
  • Atteindre la région principale : C

Avec VoiceOver :

  • Activer le rotor : (VO + U) et utiliser les flèches droite/gauche pour atteindre « Repères », puis naviguer de région en région avec flèche haut/bas

Contenu

Avec NVDA :

  • Naviguer entre les titres : H et Shift + H
  • Afficher la liste des titres : inser + F7
  • Naviguer entre les listes : L et Shift + L
  • Naviguer entre les tableaux : T et Shift + T
    • Naviguer entre les lignes d'un tableau : Ctrl + Alt + Haut et Ctrl + Alt + Bas
    • Naviguer entre les cellules d'une ligne : Haut et Bas
  • Naviguer entre les images : G et Shift + G
  • Naviguer entre les liens : K et Shift + K

Avec VoiceOver :

  • Naviguer entre les titres :
    • VO + Command + H et VO + Command + Shift + H
    • OU activer le rotor : (VO + U) et utiliser les flèches droite/gauche pour atteindre « En-têtes », puis naviguer de titre en titre avec flèche haut/bas
  • Naviguer entre les listes : VO + Command + X et VO + Command + Shift + X
  • Naviguer entre les tableaux : VO + Command + T et VO + Command + T
    • Naviguer entre les lignes d'un tableau : VO + Haut et VO + Bas
    • Naviguer entre les cellules d'une ligne : Haut et Bas
  • Naviguer entre les liens : VO + Command + L et VO + Command + Shift + L

Formulaires

  • Naviguer entre les champs avec Tab et Shift + Tab
  • Naviguer dans un groupe de boutons radio avec les touches fléchées haut/bas ou gauche/droite

Scripts

  • Activer les boutons (<button> ou role="button") avec les touches Enter et Espace
  • Pour les composants qui implémentent des modèles de conception, voir les interactions au clavier décrites sur ARIA APG Patterns :
    • Fenêtres modales
    • Disclosures
