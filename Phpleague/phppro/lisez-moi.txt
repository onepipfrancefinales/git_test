Fiches clubs
------------

Admin :          classe          => création des classes
                 rens            => création des renseignements
                 gestequipes     => valeur des renseignements
                 verif_clubs     => vérification de toutes les fiches clubs
Consultation :   consult/club.php          => consultation des fiches clubs

- D'abord il faut définir les classes c'est à dire les grandes
  parties dans lesquelles vous allez entrer les renseignements.
  (Exemple : Club, Effectif, Palmarès)

- Ensuite entrez les renseignements. Là vous allez détaillez les classes.
  (Par exemple dans club si vous voulez mettre le staff il vous faut créer
  les renseignements : président, entraineur... et les insérer dans la classe club)

- La dernière étape consiste à entrer la valeur du renseignement (Gestion des clubs).
  Pour cela il faut sélectionner le club et entrer la valeur des renseignement.
  
- Si vous ne souhaitez pas utiliser les fiches clubs, éditez "config.php"
 (remplacer le "1" par "0") ligne 23.


Classement
----------

Couleur : consult/classement.php
Barres  : consult/classement_barre

- Deux présentations pour les classements vous sont offertes : un avec les codes couleurs
  l'autre avec des barres noires séparant les équipes promues et les relégables.
  Nota : vous pouvez encadrer le premier non promus en supprimant les "//" dans le
  fichier consult/fonctions.php aux lignes 747 et 862 (je pense au classementde Ligue1...)


Mini-classement
---------------

Admin : consult/miniclass.php

- Editeur du code vous permettant d'afficher le mini-classement selon vos préférences ;-)


Buteurs
-------

- Sélectionner le buteur, pour le supprimer : cliquer sur le buteur.
- Pour la consulation, un classement équipe par équipe est maintenant disponible.


Calendrier
----------

Consultation : consult/calendrier.php


Mise à jour d'une version autre que 0.71
----------------------------------------

- Pour mettre à jour sans perdre vos données existantes :
  Sauvegardez avec phpmyadmin vos tables (seulement les données, pas la structure),
  puis chargez le fichier phpproeague.sql dans votre base (cela recréera vos tables avec la 
  bonne structure), enfin, rechargez vos données.
  NB : si vous rencontrez des problèmes avec la table 'clmnt' vous pouvez la supprimer.

- Il est fortement possible que vous rencontriez des erreurs car la structure a beaucoup changé


Statistiques
------------

Admin : admin/graph.php                  => création des graphiques
Consultation : consult/graphiques.php    => consultation des les graphiques d'un championnat

- Il faut générer les graphiques après chaque nouvelle entrée de résultats. Pour cela allez 
  dans "création de graphiques". L'exécution de ce script est assez long. Si vous rencontrez
  des problèmes à ce niveau modifiez le max_execution_time dans C:\windows\php.ini
