Expressions d'inspiration médiatique

Les mouches ont changé d'âne :
L'équipe qui menait est en train d'être menée.

La cabane est tombée sur le chien :
Le sort du match semble scellé !

Le cochon est dans le maïs :
Ca sent le roussi !

Le chat est maigre :
Le score est petit.

La jument n'a pas encore tourné le dos au foin :
Rien n'est joué !

C'est à la fin de la foire qu'on compte les bouses ! :
C'est pas encore plié pour faire les comptes.
